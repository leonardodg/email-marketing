

DEPENDENCIAS:


- Instalar e executar o GULP na pasta


> npm install -g sass


> npm install



FUNCIONAMENTO DO PROJETO PARA DESENVOLVIMENTO:

Passo 1 > Instalar Git na versão atual e adequada ao sistema:
			Link > https://git-scm.com/downloads

Passo 2 > Clonar Repositório qisat/email-marketing;

Passo 3 > Baixar e instalar Node JS na versão atual e adequada ao sistema:
			Link > https://nodejs.org/en/download/

Passo 4 > Baixar e instalar Ruby na versão atual adequada ao sistema:
			Link > https://rubyinstaller.org/downloads/

Passo 5 > Instalar Compass pela Linha de Comando:
			1. $ gem update --system
			2. $ gem install compass 

Passo 6 > Instalar Gulp e os plugins necessários pela Linha de Comando:
			1. npm install

Passo 7 > Instalar servidor/webserver;

Passo 8 > Editar host;

Passo 9 > Editar variável 'url_base' no gulpfile.js para o host selecionado;

Passo 10 > Rodar a tarefa "gulp" na Linha de Comando;

Pronto! O projeto está pronto para uso do desenvolvedor.
